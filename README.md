# 💪 Consultoria Berg Junior

Site profissional desenvolvido para apresentar serviços de **Personal Trainer online**, com foco em **emagrecimento** e preparação para o **TAF (Teste de Aptidão Física)**.

---

## 🚀 Sobre o Projeto

Este projeto é uma landing page moderna, responsiva e focada em conversão, criada para atrair novos alunos e facilitar o contato direto via WhatsApp.

O site apresenta:

* Banner com chamada de impacto
* Seção "Sobre Mim"
* Serviços oferecidos
* Formulário de contato
* Integração com WhatsApp

---

## 🖥️ Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts (Poppins e Inter)
